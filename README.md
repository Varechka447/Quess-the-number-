# Quess-the-number
Приложение «Guess the Number» - это простая игра, в которой программа загадывает число, а пользователь должен его угадать, вводя свои варианты. После каждой попытки программа сообщает, больше или меньше введённое число.

Игра позволяет пользователю:
-вводить предполагаемые числа;

-получать подсказки от программы (больше или меньше);

-взаимодействовать с программой через ввод данных;

-видеть количество сделанных попыток;

-развивать внимательность и логическое мышление.

Проект показывает работу простых алгоритмов и условных операторов в программировании.
